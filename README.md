# 📊 PhonePe Transaction Analysis Dashboard (Power BI)

## 🔹 Overview

This project presents an interactive Power BI dashboard analyzing digital transactions across multiple services such as Insurance, Loans, Money Transfer, and Recharge & Bills. The dashboard provides actionable insights into transaction trends, success rates, and failure reasons.

---

## 🔹 Features

* 📅 Dynamic Date Range Filtering
* 💰 KPI Cards (Total Amount, Total Transactions)
* 📊 Service-wise Analysis
* 📈 Trend Analysis (Date vs Amount)
* 🥧 Payment Status Distribution
* ⚠️ Failed Transaction Analysis
* 🔄 Multi-page Navigation

---

## 🔹 Tools & Technologies

* Power BI Desktop
* Power Query
* DAX
* Excel Dataset

---

## 🔹 Key Insights

* High percentage of successful transactions
* Certain services dominate transaction volume
* Monthly trends show fluctuations and peaks
* Failure reasons highlight system and user issues


## 🔹 How to Use

1. Download the `.pbix` file
2. Open in Power BI Desktop
3. Use filters and navigation tabs to explore insights

---

## 🔹 Conclusion

This dashboard demonstrates how raw transaction data can be transformed into meaningful insights for better decision-making and performance optimization.
